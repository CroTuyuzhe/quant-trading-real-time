# rt-trade-signal — 盘中趋势驱动型实时买卖决策引擎

## 简介

基于当天分钟级数据的趋势分析系统，通过 6 维因子判断未来趋势方向，给出买入/卖出/等待信号。

## 触发方式

```
buy 600900    → 检测上升趋势，提前买入
sell 000001   → 检测下跌趋势，尽快卖出
```

## 核心逻辑

**只分析当天分钟数据，不看历史交易日。**

| 用户意图 | 趋势判断 | 信号 |
|---------|---------|------|
| buy | 上升趋势正在形成 | 🟢 买入 |
| buy | 下跌/加速下跌中 | ⏳ 等待 |
| sell | 下跌趋势正在形成 | 🔴 卖出 |
| sell | 上涨/加速上涨中 | ⏳ 等待 |

## 6 维趋势因子

| 因子 | 权重 | 核心判断 |
|------|------|---------|
| 价格趋势 | 基础 | 短长期斜率方向 |
| 动量状态 | 拐点 | 三段斜率加速/减速 |
| 均线排列 | 确认 | MA5-MA20 差值变化 |
| 量能趋势 | 验证 | 放量确认趋势 |
| 反转信号 | 预判 | 见顶/见底检测 |
| 突破阶段 | 时机 | 蓄势/确认/失败 |

## 使用方法

```bash
# 单次分析
python3 scripts/rt_signal.py --code 600900 --action buy --once

# 循环监控（每30秒，最长30分钟）
python3 scripts/rt_signal.py --code 600900 --action buy --interval 30 --max-minutes 30
```

## 数据源

- **主**：新浪财经（行情 + 五档盘口 + 分钟 K 线）
- **兜底**：腾讯财经 → akshare
